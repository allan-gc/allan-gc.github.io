Impact Dynamics Simulation

overview

I simulated the dynamics of a dice in a box using Python and Jupyter Notebook. The reference frames for the dice and the box were defined and used to calculate the Lagrangian, the Euler-Lagrangian, and the impact equations. The trajectories of the system configuration variables were then plotted and animated to simulate the dynamics. The full code and report can be found on my Github.
